Second virial coefficient B₂(T) — in the browser

Type a pair potential V(r). It is compiled to a WebAssembly module in your browser (AOT) and run natively to integrate B₂ = −2π ∫₀∞ (e−V/T−1) r² dr. Because V(r) is T-independent, the same pass also returns the first two T-derivatives (analytically) and the effective repulsive exponent n_eff = −3(B₂+T·B₂′)/(2T·B₂′+T²·B₂″). Reduced units.
